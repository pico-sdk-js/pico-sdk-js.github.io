# Stats

Display statistics from a connected Raspberry Pi Pico.

## Usage

```bash
.stats
```

Output will look similar to below:

```bash
> .stats
   name            :  pico-sdk-js
   version         :  0.0.1
   running         :  main.js
   total ram       :  66036
   used ram        :  2468
   available ram   :  63568
   js engine       :  JerryScript
   js version      :  1
   js bytes        :  9880
   js peak bytes   :  11512
   js size         :  184312
```
